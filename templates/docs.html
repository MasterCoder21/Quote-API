Quote API Documentation

All of the helpful information about this API is included below, along with the different endpoints and paramaters for certain endpoints.

- GET /quotes/list

Aliases:

  • /quotes/all
  • /quotes/list-all

Parameters: None

Returns all of the quotes in the database.



- GET /quotes/count

Aliases:

  • /quotes/count-list
  • /quotes/num-list
  • /quotes/number-list

Parameters: num (int)

Usage: /quotes/count?num=(your_number)

Returns the number of quotes you asked for, no duplicates.



- GET /quotes

Aliases:

  • /quotes/
  • /quotes/id

Parameters: id (int)

Usage (only for last alias): /quotes/(id_of_quote)

Returns a single random quote or the quote with the ID you specified (only for the last alias)



- POST /quotes/add

Aliases:

  • /quotes/post

Parameters: author (str), quote (str)

Usage: /quotes/add?author=(author_name)&quote=(your_quote)

Posts a quote to the database and returns the JSON data of what you just posted.

- PUT /quotes/update

Aliases:

  • /quotes/put

Paramaters: id (int), author (str), quote (str)

Usage: /quotes/update/id?author=(author_name)&quote=(your_quote)

Updates the quote author and content corresponding to the ID you specified.

- DELETE /quotes/delete

Aliases:

  • /quotes/del
  • /quotes/remove

Paramaters: id (int)

Usage: /quotes/delete/id

Deletes the comment with the ID you specified.
